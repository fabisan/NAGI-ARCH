# 目的 
- AI(Gemini)を使って、CBT**っぽい**ことをやりたかった。 
- 折角なので日記を自然とつけれる形にしたかった。
- 普段Garmin Foreathlete 945を身につけているので、ストレススコアもGeminiに見せて記録してもらおう
- 1ヶ月間の記録をまとめて、医師に相談することの洗い出しにも使えるかも

という安直な考えから作成しました。  
Geminiに欲しい条件だけ教えて、対話形式でプロンプトを作ってもらっています。  

ちなみに命名はGeminiです。  

# 使い方
## 準備
1. ２つのチャットルームを使用します  
    ルームA：日々の記録用  
    ルームB：1週間のまとめの保管庫

1. ルームAのはじめに「**チャットルームA：初回システムプロンプト(最終確定版)**」を投稿します

1. ルームBのはじめに「**チャットルームB：初回システムプロンプト（調整版）**」を投稿します  

## 日々の使い方
1. ルームAに「**ログフォーマット**」を書き換えて好きなときに投稿します  

ログフォーマット記入例 
``` 
#SSLOG
[08:05]
[ベッド/起床後]
Neg: [0%]
Com: [おはよー]
```
日付の入力は不要だそうです  
Negは、不安感だったりイライラや悲しさなどをネガティブな感覚を主観で数値化します  
ウェアラブルウォッチを使用している場合、ストレススコアのグラフなどを添付すると、Negとグラフの値を比較して評価してくれるみたいです  

2. 毎週1回くらい、ルームAで「**週次サマリー出力プロンプト**」をそのまま投稿します  
そして「週次サマリー出力プロンプト」の出力をそのままルームBにコピー＆ペーストします

## ルームA更新作業(月1回くらい)
1. ルームAに「**ルームA：引き継ぎ用プロファイル出力プロンプト**」を投稿して、引き継ぎ用データを出力してもらいます  
1. 新しくルームAを作り直します
1. 新しいルームAのはじめに「**ルームA:初回システムプロンプト(引き継ぎ対応版)**」を投稿します
1. 次に、古いルームAで出力した引き継ぎ用データを新しいルームAに投稿して準備完了です  

## 受診前レポートの作成(必要があれば)
受診日が近づいてきたら、ルームBで **【受信直前】レポート作成用プロンプト**  を投稿します  
出力から医師に伝える必要があるものを、問診時に伝えるつもりの設計です

## 受診後
医師からのフィードバックを**医師フィードバック入力フォーマット**に入力して、ルームBに投稿します。  
ルームAは受診と同時に立て直すつもりでGeminiに作ってもらいましたが、ルームAを継続利用する場合にはルームAにも投稿してもいかもしれません
